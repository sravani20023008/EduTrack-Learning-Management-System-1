# EduTrack – Learning Management System

EduTrack is a full-stack Learning Management System developed using Python, Flask, SQLite, HTML, and Jinja2. The application supports user management, course management, enrollment, assignment creation, submission tracking, grading workflows, and search functionality.

## Tech Stack
Python, Flask, SQLite, HTML, CSS, Jinja2, SQL

## Key Features
- Role-based access for students and instructors
- Course and assignment management
- Student enrollment and submission tracking
- Grading workflow for instructors
- Dynamic search functionality
- Secure SQL queries using parameterized statements

## Project Type
Academic Database Systems Project
